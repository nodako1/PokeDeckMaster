# PokeDeckMaster

Version: 1.0.0

PokeDeckMaster はポケモンカードイベントの入賞デッキやランキングを快適に閲覧するためのシンプルなビューアアプリです。

## 公開目的
App Store 審査に必要なプライバシーポリシー/サポートページの公開 URL を提供するためのスナップショットです。過去の開発履歴は含めていません。

## 公開ページ
- プライバシーポリシー: https://nodako1.github.io/PokeDeckMaster/privacy-policy
- サポート: https://nodako1.github.io/PokeDeckMaster/support
