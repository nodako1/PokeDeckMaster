# PokeDeckMaster ドキュメント

このディレクトリには、PokeDeckMasterの公開ドキュメントが含まれています。

## 📁 ファイル構成

- `index.md` - メインのランディングページ
- `privacy-policy.md` - プライバシーポリシー
- `support.md` - サポート・お問い合わせページ
- `_config.yml` - Jekyll設定（GitHub Pages用）

## 🌐 公開URL

GitHub Pagesで公開されます：
https://nodako1.github.io/PokeDeckMaster/

## 🛠️ ローカルでの確認方法

```bash
# Jekyllをインストール（初回のみ）
gem install bundler jekyll

# docsディレクトリで実行
cd docs
bundle exec jekyll serve

# ブラウザで確認
# http://localhost:4000/PokeDeckMaster/
```

## ✏️ 編集方法

各Markdownファイルを編集してコミット・プッシュすると、GitHub Pagesが自動的に更新されます。

## 📝 OGP画像

`assets/ogp.png` に1200x630pxの画像を配置すると、SNSでシェアした際に表示されます。
