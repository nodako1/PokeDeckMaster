# 🎴 PokeDeckMaster

**ポケモンカードゲームのデッキ管理を、もっとスマートに。**

PokeDeckMasterは、ポケモンカードゲームプレイヤーのための、シンプルで使いやすいデッキ管理iOSアプリです。

---

## ✨ 主な機能

### 📱 デッキ作成・管理
カード検索から登録まで、直感的な操作でデッキを簡単に作成できます。

### 🔍 高度な検索機能
カード名、タイプ、レアリティなど、様々な条件でカードを検索できます。

### 📊 デッキ分析
デッキの構成をグラフで可視化。エネルギーバランスやタイプ分布を一目で確認できます。

### 💾 データ同期
複数のデバイス間でデッキデータを同期（将来実装予定）。

---

## 🚀 使い方

1. **App Storeからダウンロード**  
   *（リリース準備中）*

2. **デッキを作成**  
   新規デッキを作成して、お気に入りのカードを追加

3. **デッキを整理**  
   タグやカテゴリーで複数のデッキを管理

4. **対戦に持ち込む**  
   作成したデッキを確認しながら対戦を楽しむ

## ❓ よくある質問（FAQ）

### アプリは無料で使えますか？
はい、基本機能は無料でご利用いただけます。

### どのiOSバージョンに対応していますか？
iOS 15.0以降に対応予定です。

### デッキのエクスポート機能はありますか？

公式のポケモンカードゲームデータベースを参照しています。
